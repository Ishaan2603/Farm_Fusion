# 🌾 Farm Fusion – AI-Powered Smart Agriculture Platform

**Farm Fusion** is an AI-powered smart agriculture platform designed to assist farmers, agricultural professionals, and agri-enthusiasts with intelligent, data-driven farming decisions. The platform combines machine learning and deep learning models with essential agricultural resources to improve crop productivity, reduce losses, and promote sustainable farming practices.

🌐 **Live Demo:** https://farm-fusion-sigma.vercel.app/

---

## 🚀 Features

### 🌱 Crop Recommendation System

Recommend the most suitable crops based on soil and environmental conditions.

**Model:** Convolutional Neural Network (CNN)

**Input Parameters:**

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* Soil pH
* Rainfall

**Output:**

* Best-suited crop recommendations for the given conditions

---

### 🧪 Fertilizer Recommendation System

Provide optimized fertilizer suggestions to maximize crop yield and soil health.

**Model:** Convolutional Neural Network (CNN)

**Input Parameters:**

* Crop type
* Soil nutrient information

**Output:**

* Recommended fertilizer
* Nutrient composition and ratios

---

### 🍃 Plant Disease Prediction

Detect plant diseases from leaf images and provide treatment recommendations.

**Supported Crops:**

* Tomato
* Potato
* Wheat

**Model:** CNN-based Image Classification

**Input:**

* Uploaded leaf image

**Output:**

* Disease name
* Prediction confidence score
* Recommended treatment measures
